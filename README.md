# TIKTOK DOWNLOADER (NEXT.js)
      

Just paste your Tiktok URL down there to download the video without the watermark and the audio (Vocal) file seperately! Its FREE ENJOY:)



## Getting Started

Clone project and enter in folder:

```
cd songkeyfinder
```

Run npm install:

```
npm i
```

Enjoy:

```
npm run dev
```
